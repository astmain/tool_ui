# Icon 图标

U1Design 的 Icon 图标用于按钮, 表单, 导航和反馈场景. 图标保持一致的线性风格, 统一尺寸和描边, 不使用 emoji 作为结构图标.

## 尺寸控制

`.icon-xxx` 基类已经自带 `width: 1em; height: 1em`, 默认尺寸跟随父元素 `font-size`. 消费端只需在父元素(或自身)改 `font-size`, 不需要再写 `width/height`:

```html
<!-- 父元素 14px → 图标 14x14 -->
<div style="font-size: 14px"><div class="icon-menu" /></div>

<!-- 直接覆盖为 20px → 图标 20x20 -->
<div class="icon-menu" style="font-size: 20px" />
```

如果用 `<U1Icon name="xxx" />` 组件, 它走 `.u1-icon` 类, 行为同样是 `1em`, 一样通过父元素 `font-size` 控制.

## 使用原则

1. 同一层级使用同一种图标风格, 不混用线性图标和填充图标.
2. 图标按钮必须保留可访问名称, 不能只依赖视觉符号.
3. 图标和文字的间距保持 8px, 对齐到文本基线.
4. 按钮图标直接传字符串名称, 例如 <code>icon="add"</code> 或 <code>iconRight="check"</code>.
5. 未匹配到的图标名称会回退到 <code>close</code>, 避免图标区域空白.
